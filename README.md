# houselights-event-management
A modern and responsive event management landing page built with HTML, CSS, and JavaScript featuring event planning, ticket management, pricing, interactive navigation, and premium theatre-inspired UI design.
## ✨ Features

- 🎭 Modern theatre-inspired UI
- 🎟️ Interactive ticket-style components
- 📅 Event planning showcase
- 📢 Event promotion features
- 💳 Pricing section
- 📱 Fully responsive design
- 🍔 Mobile navigation menu
- ⚡ Smooth scrolling
- 🎨 Custom typography & animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- CSS Grid
- Flexbox

This project demonstrates:
